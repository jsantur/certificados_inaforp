## 📋 Descripción del Proyecto

Plataforma web desarrollada para **INAFORP – ESIN**, destinada a la verificación y consulta en línea de certificados emitidos por la institución. El sistema permite validar la autenticidad de los documentos mediante una clave única de verificación, brindando acceso rápido, seguro y confiable a la información certificada.

## 🚀 Características Principales

### 🔐 Sistema de Verificación Digital

- Acceso público a la información certificada.
